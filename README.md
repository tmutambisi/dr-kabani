# Dr. Daniel Ngombo Kabani - Official Website

This is the official website for Dr. Daniel Ngombo Kabani, International Speaker, CEO & Author.

## Project Overview

This platform is designed to:
- Support authority and legacy
- Scale revenue across multiple offer tiers
- Attract both mass-market and elite clientele
- Integrate business excellence with depth, wisdom, and purpose

## Technology Stack

- **Framework**: React with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn-ui

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd kabani-web

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Deployment

Build the project for production:

```sh
npm run build
```
